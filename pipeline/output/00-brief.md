## Projet

- **Type** : Vitrine / Portfolio artiste
- **Objectif principal** : Génération de leads — demandes de devis via le site
- **KPI** : Nombre de demandes de devis reçues via le formulaire contact
- **Approche** : Mobile-first (trafic provenant des réseaux sociaux)

### Objectifs Détaillés

1. Créer un portfolio immersif mettant en valeur 15 ans de créations murales avec une navigation par catégorie (entreprises, particuliers, événementiel, participatif)
2. Faciliter la prise de contact et la demande de devis depuis n'importe quelle page
3. Raconter le storytelling de l'artiste (parcours graffiti → design mural professionnel) pour créer une connexion émotionnelle
4. Optimiser le SEO local pour capter les recherches "fresques murales" + ville/région

## Cibles

### Persona A — Le professionnel local
- Gérant de commerce, restaurant ou entreprise en Occitanie cherchant à personnaliser son espace
- Besoin : décoration murale originale et sur mesure pour se démarquer (façade, intérieur, salle de restaurant)
- Comportement : recherche Google locale ("fresque murale restaurant Rodez"), bouche-à-oreille, recommandations

### Persona B — Le particulier créatif
- Propriétaire souhaitant une décoration unique pour sa maison (chambre d'enfant, salon, extérieur)
- Besoin : transformer un espace de vie avec une œuvre personnalisée
- Comportement : découvre via Instagram, cherche ensuite "peintre décorateur mural" sur Google, regarde le portfolio

### Persona C — L'organisateur d'événements / collectivité
- Responsable de municipalité, association sportive ou organisateur d'événements
- Besoin : fresque pour espace public, animation live painting pour événement, décoration de bâtiment communal
- Comportement : recherche de prestataires via annuaires pro, Google, recommandations institutionnelles

## Notes Stratégiques

- **Le portfolio EST le produit** : contrairement à un site vitrine classique, ici les photos des réalisations sont le principal argument de vente. L'expérience visuelle doit être premium (grandes images, transitions fluides, lazy loading performant).
- **Deux comptes Instagram = confusion** : @guihomefresquesmurales et @guihomedesignmural créent une fragmentation. Le site doit unifier la marque sous une identité cohérente. Recommandation : utiliser "IWOK" comme marque artistique forte sur le site.
- **Assets richesse = force** : +100 photos de projets réels sur 7 ans (2018-2025) couvrant 4 catégories. C'est un atout majeur — mais la qualité/résolution de chaque photo devra être vérifiée.
- **SEO local double-levier** : cibler à la fois les recherches métier ("fresquiste mural", "graffeur professionnel") et géographiques ("fresque murale Rodez", "décoration murale Aveyron").
- **Storytelling authentique** : le parcours graffiti → design mural pro est un narratif fort et différenciant. Le site doit raconter cette histoire (page À propos), pas juste lister des services.
- **Mobile-first impératif** : le trafic viendra d'Instagram/Facebook → smartphones. La galerie doit être pensée pour le swipe et le scroll vertical en priorité.
